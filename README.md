# Imersão Dados com Python II - Janeiro/2026

Este projeto é uma aplicação interativa desenvolvida em Python utilizando Streamlit, com o objetivo de explorar e visualizar dados de salários na área de tecnologia. A aplicação permite filtrar, analisar e visualizar informações relevantes a partir de um conjunto de dados real, facilitando o entendimento do mercado de trabalho em tecnologia.

## Objetivo

O principal objetivo do projeto é proporcionar uma experiência prática de análise de dados, utilizando ferramentas modernas e acessíveis. Os usuários podem interagir com os dados, aplicar filtros e visualizar gráficos dinâmicos, tornando o aprendizado mais envolvente e aplicável ao dia a dia profissional.

## Funcionalidades
- Carregamento de dados a partir de um arquivo CSV local
- Filtros interativos na barra lateral
- Visualização de gráficos e tabelas
- Análise de salários por diferentes categorias

## Como executar localmente

1. Clone o repositório:
   ```bash
   git clone <URL-do-repositorio>
   ```
2. Instale as dependências:
   ```bash
   pip install -r requirements.txt
   ```
3. Execute a aplicação:
   ```bash
   streamlit run app.py
   ```
