# Imersão Dev Alura - Aula 04

Aplicação Streamlit para visualizar e analisar dados de salários em tecnologia.

## Como usar
1. Instale as dependências:
   ```bash
   pip install -r requirements.txt
   ```
2. Execute:
   ```bash
   streamlit run app.py
   ```
